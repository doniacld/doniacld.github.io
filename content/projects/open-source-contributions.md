A fast, configurable, extensible, flexible, and beautiful linter for Go.
- Implemented new linting rules
- Added Docker image to the release

[github.com/mgechev/revive](https://github.com/mgechev/revive)

### TinyGo Drivers
Hardware drivers for TinyGo - the Go compiler for microcontrollers.

[github.com/tinygo-org/drivers](https://github.com/tinygo-org/drivers)

### GoBadge
Run TinyGo programs on an Adafruit board. I've been granted organizer status and can review and merge PRs.

[github.com/tinygo-org/gobadge](https://github.com/tinygo-org/gobadge)
