Building beautiful terminal applications with the Charm CLI toolkit, presented at GopherCon 2024.

## Abstract

Terminal applications don't have to be boring! With Charm CLI tools, you can build glamorous TUIs. This talk covers:

- Introduction to Charm CLI ecosystem (Bubble Tea, Lip Gloss, etc.)
- Building a habit tracker from scratch
- Adding style and animations
- Best practices for Go CLI development
